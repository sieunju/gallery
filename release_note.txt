### Introduction
> It is a library that helps you retrieve images from the gallery and edit them all at once.
> Library efficiency is divided into Multi-Modules.
ex.) core, edit, ui

### Additional details
> We added core-rx, which can easily use core module as ReactiveX!
> We added core-coroutines that can easily use core modules as Coroutines!
